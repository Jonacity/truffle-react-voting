# Voting dApp

## Install dependencies

```
npm install

// Install truffle
npm install truffle

// Install ganache
npm install ganache-cli

// Install eth-gas-reporter
npm install eth-gas-reporter
```

## Run unit tests

```
// Run ganache
ganache-cli

// Run migrations
truffle migrate

// Run tests
truffle test
```

> 27 passing tests

1 file: `VotingTest.js`

## Start app

```
// Run ganache
ganache-cli

// Run migrations
truffle migrate

// Start react app
npm start
```
